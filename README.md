## 🚀 Instalasi & Setup

### 🔧 Requirement

- PHP >= 8.2
- Composer
- MySQL / MariaDB
- Node.js + NPM
- Laravel 12.x

### ⚙️ Langkah Instalasi

```bash
git clone https://github.com/username/nama-repo.git
cd nama-repo
cp .env.example .env
composer install
php artisan key:generate
npm install && npm run dev
php artisan migrate --seed
php artisan serve
```
